PROTOTIPO AR - GUIA 3

Contenido:
- index.html  -> archivo principal
- hiro_marker.jpg -> marcador que debe ver la camara

Que hace:
- Muestra un prototipo WebAR con un recipiente, un nivel de agua y una grafica h(V).
- La idea didactica es que el estudiante observe que si el recipiente se angosta hacia arriba,
  la grafica de altura contra volumen se vuelve mas empinada.

Como usarlo rapido:
1. Cree un repositorio publico en GitHub.
2. Suba index.html y hiro_marker.jpg.
3. En Settings > Pages, active Deploy from a branch > main > /(root).
4. Espere que GitHub le genere la URL del sitio.
5. Abra la URL desde el celular.
6. Permita acceso a la camara.
7. Muestre hiro_marker.jpg impreso o en otra pantalla.
8. Apunte el celular al marcador.

Notas:
- El QR no reemplaza el marcador. El QR solo abre la pagina.
- El marcador es hiro_marker.jpg.
- Si desea, despues se pueden crear dos versiones mas del mismo prototipo:
  recipiente cilindrico y recipiente que se ensancha hacia arriba.
